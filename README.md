# VSN (Vision Signal Network)

VSN is a modular quantitative crypto analysis framework focused on detecting pump, distribution, and dump probability on MEXC.

## Vision

Build a Formula Brain that combines statistical models, probabilistic inference, and market structure into one decision engine.

## Planned Modules

- Data Engine
- MEXC Client
- Pump Scanner
- Formula Labs
- Transition Engine
- Regime Engine
- Monte Carlo Engine
- Bayesian Brain
- Central Brain
- Decision Engine
- FastAPI Backend
- React Dashboard

## High Level Architecture

Data Source
    -> Data Engine
    -> Formula Labs
    -> Transition Engine
    -> Regime Engine
    -> Monte Carlo
    -> Bayesian Brain
    -> Central Brain
    -> Decision Engine
    -> API / Dashboard
